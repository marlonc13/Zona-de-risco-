MELHORIAS IMPLEMENTADAS

1. Mapa em estilo Google Maps
- Barra superior flutuante
- Filtros por tipo de alerta
- Botão de localização
- Botão rápido para criar alerta
- Visitantes conseguem visualizar alertas sem login

2. Alertas colaborativos
- Segurar no mapa cria alerta
- Criar alerta exige login Google
- Tipos: Alagamento, Deslizamento, Bloqueio, Acidente, Outro
- Alerta sem foto dura 30 minutos
- Alerta com foto dura 1 hora
- Os alertas são salvos no Firebase Firestore
- Fotos são salvas no Firebase Storage

3. Notificações
- Quando chega um novo alerta com o app aberto, uma notificação local é disparada
- Para notificação chegar com app fechado, depois precisa configurar push notification com servidor/Cloud Functions

4. Login
- Login com Google
- Botão para continuar como visitante
- Usuário visitante vê alertas, mas não cria
- Usuário logado pode criar alertas e acessar perfil
- Se já houver sessão logada, o app pede biometria na entrada

5. Perfil
- Tela de perfil
- Editar nome
- Ver email
- Trocar foto de perfil pela câmera ou galeria
- Trocar senha opcional
- Sair da conta

IMPORTANTE

Você ainda precisa configurar no Firebase/Google Cloud os Client IDs do Google no arquivo:
app/login.js

Troque:
webClientId: 'COLE_SEU_WEB_CLIENT_ID_AQUI.apps.googleusercontent.com'
androidClientId: 'COLE_SEU_ANDROID_CLIENT_ID_AQUI.apps.googleusercontent.com'

Também precisa liberar regras no Firebase:
- Firestore para leitura pública dos alertas
- Firestore para escrita apenas autenticada
- Storage para upload de imagens por usuários autenticados

Depois rode:
npm install
npx expo start --tunnel
