Para o botão "Entrar com Google" funcionar de verdade, troque estes campos em app/login.js:

const WEB_CLIENT_ID = 'COLE_SEU_WEB_CLIENT_ID_AQUI.apps.googleusercontent.com';
const ANDROID_CLIENT_ID = 'COLE_SEU_ANDROID_CLIENT_ID_AQUI.apps.googleusercontent.com';

Você pega esses IDs no Google Cloud/Firebase:
1. Firebase Console > Authentication > Sign-in method > ativar Google.
2. Google Cloud Console > APIs e Serviços > Credenciais.
3. Criar/usar OAuth Client ID do tipo Web para WEB_CLIENT_ID.
4. Criar OAuth Client ID do tipo Android para ANDROID_CLIENT_ID.
5. No Android, precisa configurar package name e SHA-1 do app.

Enquanto esses IDs estiverem com "COLE_SEU...", o app abre normalmente como visitante, mas o Google não autentica.
